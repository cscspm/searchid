One-Time Setup Instructions
①
Create Google Sheet
Go to sheets.google.com → New blank sheet → copy the Sheet ID from the URL
②
Open Apps Script
Go to script.google.com → New Project → paste the entire google_apps_script.js content
③
Paste Sheet ID
In the script, find line: const SHEET_ID = '' → paste your Sheet ID between the quotes
④
Deploy as Web App
Click Deploy → New Deployment → Web App → Execute as: Me → Access: Anyone → Deploy → copy URL
⑤
Paste URL above

⑥
Share state team URL

Backup & Restore

Export all your stored data as a single backup file. Import it on any device or browser to restore everything instantly.

Save this file to Google Drive, WhatsApp to yourself, or any safe location.

Step 2 — Restore

Import Backup File

Select the .json backup file exported from this tool. All data will be restored exactly as it was — on this device, any device, or after clearing browser data.

Click or drag · .json backup file
How to Use on Mobile / Another Device
①
Export backup on this device
Click "Export All Data" above. A .json file downloads to your laptop.
②
Send to yourself
Upload the .json to Google Drive, WhatsApp to yourself, or email it to your phone.
③
Open the HTML tool on mobile
Open the csc_processor_v7.html file from Google Drive in Chrome on your phone.
④
Import the backup
Go to Backup & Restore → Import the .json file. All data restored in seconds.
What is included in the backup:
✓ Master Blocked/Unblocked IDs (all entries)
✓ App Stage data (all CSC IDs + stage numbers)
✓ EKYC data (all unique IDs + dates)
✓ Fraud IDs (all flagged entries with details)
✓ Unblock History (all dates you processed IDs)
✓ Permanently Closed history (all dates)

What is NOT included: Today's daily verified/non-verified uploads (those are session-only, upload fresh each day).
